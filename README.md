AI Startup Idea Generator
Overview

This is one of my first Python projects. It generates a startup idea based on the industry selected by the user.

Features
Interactive command-line interface
Multiple industries to choose from
Personalized startup ideas
Handles unsupported industries with a friendly message
Technologies Used
Python
Skills Practiced
Variables
User input (input())
Conditional statements (if, elif, else)
String handling
Program logic
What I Learned

Building this project helped me understand how programs make decisions based on user input. It strengthened my understanding of Python fundamentals and introduced me to writing interactive applications.
